# Expense Tracker (Python)

A console-based expense tracker that stores data in a CSV file and provides
monthly and category-wise summaries.

## Features
- Add expense (date, category, amount, note)
- View all expenses
- Monthly summary
- Category-wise summary
- Clear all data (optional)

## Concepts Used
- File I/O with CSV
- Basic data aggregation
- Functions and control flow

## How to Run
```bash
python main.py
